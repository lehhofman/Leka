2. CSS: Flexbox e Alinhamento Complexo

Domine o Flexbox para layouts dinâmicos e responsivos.

Crie um layout com três itens alinhados no centro usando Flexbox.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
.item {
  margin: 10px;
  padding: 20px;
  background-color: lightcoral;
}

Dica: Use 'justify-content' e 'align-items' para centralizar.
