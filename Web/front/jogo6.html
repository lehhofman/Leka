Learning Path

Aqui está seu caminho de aprendizado: HTML, CSS e JavaScript.

1. HTML: Elementos e Atributos

Aprenda sobre os principais elementos HTML e seus atributos.

Passo 1:

Adicione uma tabela à sua página HTML.

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>João</td>
    <td>25</td>
  </tr>
</table>

Dica: Use <table>, <tr>, <th> e <td> para criar a tabela.

2. CSS: Estilizando Elementos

Aprenda a aplicar estilos básicos a elementos HTML.

Passo 1:

Estilize um botão para parecer mais atraente.

button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
}

Dica: Use propriedades como 'background-color' e 'border-radius'.

Próximo Módulo

3. JavaScript: Estruturas de Controle

Entenda como usar estruturas de controle como if e loops.

Passo 1:

Crie uma função que verifica se um número é par ou ímpar.

function verificarPar(numero) {
  if (numero % 2 === 0) {
    return 'Par';
  } else {
    return 'Ímpar';
  }
}

Dica: Use o operador '%' para verificar a divisibilidade.

Próximo Módulo

4. Manipulação do DOM com JavaScript

Aprenda a modificar o conteúdo e estilo dos elementos na página.

Passo 1:

Altere o texto de um cabeçalho ao clicar em um botão.

<h2 id="titulo">Título Original</h2>
<button onclick="mudarTitulo()">Alterar Título</button>

<script>
function mudarTitulo() {
  document.getElementById('titulo').innerText = 'Novo Título!';
}
</script>

Dica: Use 'innerText' para mudar o texto de um elemento.
